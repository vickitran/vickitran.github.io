Implementing Fairness in Machine Learning

Acknowledgements:

The following project is based on the amazing work done by Fair Kernel Learning

Motivation:

Algorithm predictive modelling is utilized in multiple industries enhancing or replacing expert human decision making. Some models have been criticized for perpetuating inequality hidden in the underlying data. A notable example is Propublica’s article : Machine Bias, a detailed investivgation highlighting a racial bias in a crime predicting software tool. There’s a large and exciting academic research centered around this topic: Fairness, Accountability, and Transparency in Machine Learning .

This is my own small contribution to make cutting edge research accessible for machine learning enthusiasts, who may not have as much free time as I do.

Summary:

The paper's scientific contribution is divided into three sections. The authors performed all experiements in MATLAB. I've written a Python equivalent here!

  • Fair Linear Regression
  • Fair Kernel Regression
  • Fair Dimensionality Regression
In the next sections, I'll walk through each concept with a tiny bit of math and a worked example with HR sample data.

The techniques aim to minimize disparate impact, in which certain individuals belonging to a protected group (race,sex, religion) receive unfavourable treatment more often than non-protected groups.

As mentioned in the paper, a model is totally fair
if the predictions are statistically independent of the sensitive (protected features)

—(Perez-Suay 2017)
