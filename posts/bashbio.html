Bash and Bioinformatics - A Self Guided Tour

I have always viewed Bash to be a mythical black box commanded by consonants. Here's my ongoing attempt to break that misconception.

ROSALIND is a collection of Bioinformatics themed programming puzzles. Each puzzle also contains a short description of the puzzle's bioinformatic concept. Based on this article, a bioinformatician's toolkit typically includes Python, R and Bash. ROSALIND serves as an interactive textbook to learn BASH and biology at the same time.

In this post, I will be keeping a record of all the useful commands I have learned while solving various puzzles.

Commands + Description:

wc -{x}

Count the number of newlines, word count, characters, bytes in a given input stream. -{x} specifies what is to be counted.

Neat Trick

wc can be used to count the number of files in a directory.
ls {directory} | wc -l

tr {to_be_replaced} {replacement}

Replace or remove in a given input stream.

Neat Trick

tr can change strings from lower case to upper case and vice versa
tr [:lower] [:upper] < {input}
