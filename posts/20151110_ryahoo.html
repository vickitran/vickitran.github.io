Using R to Pull Yahoo Stock Finance Symbols

This little side project was born out of my hours of searching for a simple Excel list of stock symbols sorted by sector. I decided to it would be easier to write my own script to do just that.

What I used/learned:
  • Magrittr
  • Rvest
  • XPath

Magrittr is a magical operator (%>%) that forwards the result of an expression into another expression. It makes R code easy to read and understand. I used it extensively with rvest’s built in functions to parse through the html document.

Rvest is a great web scraping library built for R, comparable to Python’s beautifulsoup. I was inspired to write this script based on this blog post. I originally chose R because MATLAB slowed down my computer to a crawl and I wanted fast prototyping.

I used a combination of XPath and CSS Selectors. CSS selectors were useful in picking general elements, however XPath offered more refinement when looking for hyperlinks.

General Method: The script starts off at the base url (http://biz.yahoo.com/p/) where the sectors are listed. It collects the hyperlinks of each sector, listed as sector_html. Then it uses the same technique to collect the hyperlinks of each industry in the sector, listed as industry_html. Lastly, it will go through each industry and collect the classified stock symbols.
